## Overview

Syra's Market Research feature provides comprehensive intelligence gathering and analysis for cryptocurrency markets. Using advanced AI models and multi-source data aggregation, it delivers executive-level summaries and actionable insights.

## Research Capabilities

### AI-Powered Web Search

Leverage EXA AI for dynamic web searches across the latest crypto news and market analysis.

```javascript
const response = await fetch('https://api.syraa.fun/v2/exa-search?query=latest+news+on+Nvidia', {
  headers: { 'Payment-Request': paymentHeader }
});

const { query, results, autopromptString } = await response.json();
```

**Key Features:**
- 10 results per search with highlights
- Up to 4,000 characters per highlight
- Auto-type detection (neural/keyword)
- Relevance scoring

### Research Dashboard

The internal research system allows for persistent storage and AI-powered summarization of research findings.

#### Store Research

```javascript
// Save research to dashboard
await fetch('https://api.syraa.fun/v1/internal/research-store', {
  method: 'PUT',
  headers: { 
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    payload: {
      panels: { /* research panels */ },
      customXSearch: { /* search results */ },
      deepResearch: { /* deep research */ }
    }
  })
});
```

#### Retrieve Research

```javascript
// Load saved research
const response = await fetch('https://api.syraa.fun/v1/internal/research-store', {
  headers: { 'Authorization': `Bearer ${apiKey}` }
});

const { payload, savedAt } = await response.json();
```

## Integration Example

```javascript
import { PaymentClient } from '@faremeter/x402-client';

class MarketResearch {
  constructor(apiKey) {
    this.apiKey = apiKey;
    this.paymentClient = new PaymentClient({
      network: 'solana',
      wallet: yourWallet
    });
  }

  async searchNews(query) {
    const response = await this.paymentClient.fetch(
      `https://api.syraa.fun/v2/exa-search?query=${encodeURIComponent(query)}`
    );
    
    if (response.ok) {
      return await response.json();
    }
  }

  async saveResearch(researchData) {
    const response = await fetch(
      'https://api.syraa.fun/v1/internal/research-store',
      {
        method: 'PUT',
        headers: {
          'Authorization': `Bearer ${this.apiKey}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({ payload: researchData })
      }
    );
    
    return await response.json();
  }

  async generateSummary(researchData) {
    const response = await fetch(
      'https://api.syraa.fun/v1/internal/research-resume',
      {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${this.apiKey}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify(researchData)
      }
    );
    
    const { resume } = await response.json();
    return resume;
  }
}

// Usage
const research = new MarketResearch(process.env.SYRA_API_KEY);
const results = await research.searchNews('Solana DeFi ecosystem growth');
const summary = await research.generateSummary({ customXSearch: results });
```
