## Telegram Bot Support

Our [Telegram bot](https://t.me/syra_trading_bot) is the fastest way to get help and access Syra's features.

### Quick Start

1. Open the [Syra Trading Agent Bot](https://t.me/syra_trading_bot)
2. Press **Start**
3. Use `/help` to see all available commands

### Useful Commands

| Command | Description |
|---------|-------------|
| `/start` | View available commands |
| `/signal bitcoin` | Get latest BTC trading analysis |
| `/list` | Show supported tokens |
| `/news BTC` | Get latest BTC-related news |
| `/top_mention today` | Most-discussed tokens today |
| `/docs` | Open documentation |
| `/feedback` | Send suggestions or issues |

<Tip>
Use the `/feedback` command in the Telegram bot to report issues or suggest improvements directly to the team.
</Tip>
