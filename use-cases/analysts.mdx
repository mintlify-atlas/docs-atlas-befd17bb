## Token Research Framework

Comprehensive due diligence workflow for new token analysis:

### Phase 1: Initial Screening

<AccordionGroup>
  <Accordion title="Safety Check (Solana Tokens)">
    **Rugcheck Analysis:**
    ```bash
    GET /token-report?address={token_address}
    ```

    **Red Flags:**
    - Risk score greater than 50
    - Unverified contract
    - Mint authority not revoked
    - Freeze authority enabled
    - Low liquidity (less than $50k)

    **Green Flags:**
    - Risk score less than 20
    - Verified on Rugcheck
    - Mint authority revoked
    - No freeze authority
    - Deep liquidity (greater than $500k)

    <Warning>
      Never invest in tokens with risk scores >70. These are likely scams or extremely risky.
    </Warning>
  </Accordion>
  
  <Accordion title="Holder Analysis">
    **Bubblemaps Concentration:**
    ```bash
    GET /bubblemaps/maps?address={token_address}
    ```

    **Analyze:**
    - Top 10 holder concentration (ideally less than 40%)
    - Insider/team holdings distribution
    - Whale wallet patterns
    - Exchange holdings vs. retail

    **Warning Signs:**
    - Single holder greater than 20%
    - Top 10 greater than 60%
    - Unusual clustering patterns
    - Large unlocked team allocations
  </Accordion>
  
  <Accordion title="Market Presence">
    **DEXScreener & Jupiter:**
    ```bash
    GET /dexscreener
    GET /trending-jupiter
    ```

    **Check:**
    - Trading volume trends
    - Liquidity depth
    - Price stability
    - Community takeover status
    - Boosted/promoted status

    **Healthy Signs:**
    - Consistent daily volume
    - Growing liquidity
    - Organic price discovery
    - No suspicious pumps/dumps
  </Accordion>
</AccordionGroup>

### Custom Research Reports

Build comprehensive reports by combining endpoints:

```python
# Python example: Generate token research report
import requests

token_address = "{solana_token_address}"

# Gather all data
safety = requests.get(f"https://api.syraa.fun/token-report?address={token_address}")
holders = requests.get(f"https://api.syraa.fun/bubblemaps/maps?address={token_address}")
godmode = requests.get(f"https://api.syraa.fun/token-god-mode?tokenAddress={token_address}")
kol = requests.get(f"https://api.syraa.fun/x-kol?address={token_address}")

# Generate report
report = generate_markdown_report(safety, holders, godmode, kol)
save_report(report, f"research_{token_address}.md")
```
