# **Syra**

### The Intelligence Layer for Autonomous Trading Agents

**Syra** is an intelligent, research-driven **AI Trading Intelligence Agent** for the Solana ecosystem. Built to help traders, analysts, and developers make smarter, data-based decisions by combining real-time market data, on-chain signals, narrative intelligence, and structured AI-driven research.

<Note>
  Syra is designed as **an intelligence layer**, not just a signal bot. It enhances understanding with structured research, risk-aware perspectives, and learning-oriented explanations.
</Note>

## What Makes Syra Different?

Syra delivers actionable trading intelligence through multiple channels:

- **Real-time market data** — Live technical and contextual indicators
- **On-chain activity signals** — Holder flows, smart money tracking, DEX activity
- **Narrative & sentiment intelligence** — News aggregation, trends, and market mood analysis
- **Structured AI-driven research** — Clear, actionable insights with confidence levels and reasoning

## Quick Start Options

<Tabs>
  <Tab title="Telegram Bot">
    Get started in seconds with the Telegram bot:

    1. Open [Syra Trading Agent Bot](https://t.me/syra_trading_bot)
    2. Press **Start**
    3. Use `/list` to view supported tokens
    4. Try `/signal bitcoin` for a live analysis

    **Popular Commands:**
    - `/signal bitcoin` — Get latest BTC trading analysis
    - `/news BTC` — Get latest BTC-related news
    - `/top_mention today` — Most-discussed tokens today
  </Tab>
  
  <Tab title="API Access">
    Access Syra intelligence programmatically:

    ```bash
    # Get trading signal for Bitcoin
    curl https://api.syraa.fun/signal?token=bitcoin

    # Get latest crypto news
    curl https://api.syraa.fun/news?ticker=BTC

    # Get smart money tracking data
    curl https://api.syraa.fun/smart-money
    ```

    <Info>
      Most API endpoints use x402 pay-per-use pricing. See [API Reference](/api-reference/overview) for details.
    </Info>
  </Tab>
  
  <Tab title="MCP Server">
    Use Syra in Cursor, Claude Desktop, or other AI assistants:

    ```bash
    cd mcp-server
    npm install
    npm run build
    ```

    Add to your MCP config and ask for "crypto news" or "trading signals" directly in your AI chat.
  </Tab>
</Tabs>
