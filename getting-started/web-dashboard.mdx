The Syra web dashboard provides a comprehensive interface for exploring trading signals, market insights, analytics, and leaderboards.

## Access the Dashboard

The main Syra dashboard is available at:

**[syraa.fun](https://www.syraa.fun/)**

## Features

### Signals Dashboard

View and explore AI-generated trading signals:

- **Real-time Signals** — Latest AI trading analysis for supported tokens
- **On-chain Verified Signals** — Signals stored on Solana blockchain for transparency
- **Historical Performance** — Track signal accuracy over time
- **Filter & Search** — Find signals by token, date, or performance

### Market Insights

Access deep market intelligence:

- **Research Reports** — AI-powered research on crypto topics
- **Sentiment Analysis** — 30-day market mood tracking
- **News & Events** — Curated crypto news and upcoming events
- **Trending Headlines** — What's moving the market

### Analytics

Comprehensive market data:

- **Smart Money Tracking** — Follow institutional flows
- **DEXScreener Data** — Token profiles and community takeovers
- **Jupiter Trending** — Top tokens on Jupiter DEX
- **Correlation Analysis** — Asset correlation matrices

### Leaderboard

Track performance and rankings:

- Signal accuracy leaderboards
- Community performance stats
- Top tokens by various metrics

### User Profile

Personalized experience:

- View your signal history
- Track your watchlist
- Manage preferences

## Wallet Integration

Connect your Solana wallet to:

- Access premium features
- Pay for API calls with x402
- Interact with on-chain signals
- Participate in the prediction game

<Info>
Wallet connection uses industry-standard Solana wallet adapters supporting Phantom, Solflare, and other popular wallets.
</Info>
