Access AI-powered trading intelligence and market analysis directly from Telegram. The Syra bot provides real-time signals, news, sentiment analysis, and research without leaving your chat.

## Quick Start

<Steps>
  <Step title="Open the Bot">
    Visit [@syra_trading_bot](https://t.me/syra_trading_bot) on Telegram
  </Step>
  
  <Step title="Start Conversation">
    Press the **Start** button or send `/start` to activate the bot
  </Step>
  
  <Step title="Try a Command">
    Send `/signal bitcoin` to get your first AI trading signal
  </Step>
</Steps>

## Available Commands

The Syra bot supports the following commands:

| Command | Description |
|---------|-------------|
| `/start` | View welcome message and available commands |
| `/signal <token>` | Get AI-generated trading signal with entry/exit recommendations |
| `/list` | Show all supported tokens |
| `/news <ticker>` | Get latest news for a specific token (e.g. `/news BTC`) |
| `/top_mention <period>` | Most-discussed tokens (use `today`, `week`, etc.) |
| `/docs` | Open the full documentation |
| `/feedback` | Send suggestions or report issues |

## Examples

### Get Trading Signal

```
/signal bitcoin
```

Returns AI-driven analysis including:
- Market overview (price, volume, volatility)
- Technical indicators (RSI, MACD, SMA, EMA, Bollinger Bands)
- Entry/exit recommendations
- Risk context and confidence levels

### Check Latest News

```
/news ETH
```

Get curated Ethereum news and market sentiment.

### Find Trending Tokens

```
/top_mention today
```

Discover which tokens are being discussed most on social media and news.

## What You Get

Each signal includes:

- **Market Overview** — Current price, volume, volatility, and trend strength
- **Technical Analysis** — Multiple indicators combined into actionable insights
- **Action Perspectives** — Key support/resistance levels and momentum bias
- **Risk Assessment** — R/R awareness and exposure considerations
- **AI Confidence** — Sentiment interpretation and signal strength

## Tips

<Tip>
The bot is designed for **educational and research purposes**. Signals include explanations and context to help you understand market conditions, not just blind execution recommendations.
</Tip>

<Warning>
Always do your own research. Syra provides intelligence to enhance understanding, not replace decision-making.
</Warning>
