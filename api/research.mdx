## Overview

The Research API provides internal research tools for the Syra platform, including research data persistence, retrieval, and AI-powered summarization using Jatevo.

## Authentication

Research endpoints require **API key authentication** (not x402 payment).

## Endpoints

### Store Research Data

```
PUT /internal/research-store
```

Persist research data to database.

#### Request Body

```json
{
  "payload": {
    "panels": {},
    "customXSearch": {},
    "deepResearch": {},
    "browse": {}
  }
}
```

#### Response

```json
{
  "ok": true,
  "savedAt": "2026-03-03T10:00:00.000Z"
}
```

### Load Research Data

```
GET /internal/research-store
```

Retrieve latest saved research data.

#### Response

```json
{
  "payload": {
    "panels": {},
    "customXSearch": {},
    "deepResearch": {},
    "browse": {},
    "savedAt": "2026-03-03T10:00:00.000Z"
  },
  "savedAt": "2026-03-03T10:00:00.000Z"
}
```

### Generate Research Summary

```
POST /internal/research-resume
```

Generate AI-powered executive summary of research findings using Jatevo.

#### Request Body

```json
{
  "panels": {
    "panel1": {
      "lastQuery": "Solana ecosystem analysis",
      "data": {
        "result": "Research findings..."
      }
    }
  },
  "customXSearch": {
    "lastQuery": "Latest Solana news",
    "data": {
      "result": "X search results..."
    }
  },
  "deepResearch": {
    "lastQuery": "DeFi trends",
    "data": {
      "content": "Deep research content..."
    }
  },
  "browse": {
    "lastQuery": "https://solana.com/news",
    "data": {
      "result": "Extracted web content..."
    }
  }
}
```

#### Response

```json
{
  "resume": "# Executive Research Summary\n\n## Key Insights\n- Solana ecosystem growing rapidly...\n- DeFi TVL increased 45% QoQ...\n\n## Strategic Takeaways\n- Strong institutional interest...\n\n## Risks and Opportunities\n- Network congestion during peak usage...\n\n## Recommendations\n- Monitor validator performance...",
  "truncated": false
}
```

## Research Summary Generation

The AI summarization uses **Jatevo** with the following configuration:

```javascript
const result = await callJatevo(messages, {
  max_tokens: 8192,
  temperature: 0.4
});
```

### System Prompt

```
You are an expert analyst. Given the following latest research findings 
(from X search, deep research, and browse), produce a concise executive 
resume/summary in clear sections. Focus on: key insights, strategic 
takeaways, risks or opportunities, and actionable recommendations for 
the Syra project. Use bullet points and short paragraphs. Write in English.
```

## Research Data Structure

### Panels

Multiple research panels with queries and results:

```json
{
  "panelId": {
    "lastQuery": "Query string",
    "data": {
      "result": "Research findings"
    }
  }
}
```

### Custom X Search

X/Twitter search results:

```json
{
  "lastQuery": "Search query",
  "data": {
    "result": "Tweet analysis and findings"
  }
}
```

### Deep Research

In-depth research content:

```json
{
  "lastQuery": "Research topic",
  "data": {
    "content": "Comprehensive research content"
  }
}
```

### Browse

Web page extraction:

```json
{
  "lastQuery": "https://example.com",
  "data": {
    "result": "Extracted web content"
  }
}
```

## Content Truncation

Browse results are truncated if too large:

```javascript
const text = typeof r === "string" && r.length < 50000 
  ? r 
  : (typeof r === "string" 
    ? r.slice(0, 20000) + "…" 
    : JSON.stringify(r).slice(0, 20000) + "…");
```

Research blob sent to AI is limited to **120,000 characters**.

## Error Handling

### No Content to Summarize

```json
{
  "error": "No research content to summarize",
  "hint": "Send panels, customXSearch, deepResearch, or browse with result content."
}
```

### Internal Server Error

```json
{
  "error": "Internal server error",
  "message": "Error details"
}
```

## Code Example

```javascript
// Store research data
const storeResponse = await fetch('https://api.syraa.fun/internal/research-store', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    payload: {
      panels: { /* research panels */ },
      customXSearch: { /* X search */ },
      deepResearch: { /* deep research */ },
      browse: { /* web content */ }
    }
  })
});

const { savedAt } = await storeResponse.json();
console.log(`Research saved at ${savedAt}`);

// Generate summary
const summaryResponse = await fetch('https://api.syraa.fun/internal/research-resume', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify(researchData)
});

const { resume, truncated } = await summaryResponse.json();
console.log(resume);
if (truncated) console.warn('Summary was truncated');
```

## Best Practices

1. **Aggregate Data**: Combine multiple research sources for comprehensive summary
2. **Clear Queries**: Use descriptive lastQuery fields for context
3. **Content Limits**: Keep browse results under 50KB for best results
4. **Regular Saves**: Persist research frequently to avoid data loss
5. **Check Truncation**: Monitor truncated flag in summaries
